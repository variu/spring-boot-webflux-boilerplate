# Overview

* Build a environment to focus on development
* Multi module using gradle

# Dev Env

- Spring Boot 2.6.3
- Java 17
- Postgres 14.2
- Redis 6.2

# Prerequirement

- docker & docker-compose

# Init

```
cd docker-compose
docker-compose up -d
```
